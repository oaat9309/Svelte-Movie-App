## Project Details

### Application

- SvelteKit을 통한 프로젝트 구축

- {goto}를 활용한 라우팅

- SvelteKit:prefetch 활용

### CSS

- global CSS 적용

- Google Font 적용

- Svelte Transition을 활용한 애니메이션 적용

### API

- TMDB API 활용

- .env 파일을 통한 API key 보안화

### Distribution

- Netlify를 통한 배포

### Misc

- ESLint / Prettier를 활용한 코드 관리
